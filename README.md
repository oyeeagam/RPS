# RPS

Learning by implementing: created a console based rock-paper-scissors game to understand the intricacies of javascript methods functionalities and developer tools specifically (console tab).

~ user gets prompted to enter between rock,paper,scissors 
    (handled case sensitivity)
~ the program uses javascript methods to generate a valid computer choice of its hand
~ the game is played for 5 rounds 

    the result is logged in console tab of the project document 

Learning outcomes : 
    1. javascript methods
    2. console tab developer tool
    3. git commands
    4. version control
    5. logic to implementation
